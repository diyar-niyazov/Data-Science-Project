# Video Game Sales Correlation Calculator

A Java program that parses video game sales data from a CSV file and calculates the Pearson correlation coefficients between numeric sales fields.

---

## Features

- Parses CSV files with proper handling of quoted commas.
- Stores games as objects with numeric and categorical fields.
- Aggregates data for numeric and categorical categories.
- Calculates correlation coefficients between numeric fields.
- Outputs correlation matrix for analysis.
